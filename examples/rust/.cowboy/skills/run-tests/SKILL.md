To validate this Rust crate:

1. Run `cargo build` and fix any compile errors first.
2. Run `cargo test` and read the output.
3. Report which tests passed/failed; if a test fails, show the relevant output.

Prefer running these via the `shell` tool. Use `cowboy patch show` to review
any edits you made before finishing.
